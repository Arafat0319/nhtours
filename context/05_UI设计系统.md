## 设计原则

### WeTravel 风格

后台管理系统采用 WeTravel 风格设计：
- **简洁明亮**: 白色背景 + 青色主色调
- **卡片布局**: 信息块使用卡片承载
- **清晰层级**: 通过阴影和间距区分层级

### 技术选型

| 技术 | 用途 |
|------|------|
| Tailwind CSS | 样式框架 |
| Heroicons | SVG 图标 |
| Quill.js | 富文本编辑器 |

## 色彩系统

### 主色调

| 名称 | HEX | Tailwind | 用途 |
|------|-----|----------|------|
| WeTravel Cyan | `#00D1C1` | `wetravel-cyan` | 主要按钮、链接、强调 |
| WeTravel Cyan Hover | `#00B8A9` | `wetravel-cyan-dark` | 悬停状态 |
| WeTravel Green | `#4ADE80` | `wetravel-green` | 成功状态 |

### 中性色

| 名称 | HEX | Tailwind | 用途 |
|------|-----|----------|------|
| 背景色 | `#F9FAFB` | `gray-50` | 页面背景 |
| 卡片背景 | `#FFFFFF` | `white` | 卡片、面板 |
| 边框色 | `#E5E7EB` | `gray-200` | 分割线、边框 |
| 正文色 | `#374151` | `gray-700` | 正文文字 |
| 次要文字 | `#6B7280` | `gray-500` | 辅助说明 |
| 标题色 | `#111827` | `gray-900` | 标题 |

### 状态色

| 状态 | 颜色 | 用途 |
|------|------|------|
| 成功 | `green-500` | 完成、已付款 |
| 警告 | `yellow-500` | 待处理、提醒 |
| 错误 | `red-500` | 错误、失败 |
| 信息 | `blue-500` | 提示信息 |

### Tailwind 配置

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        'wetravel-cyan': '#00D1C1',
        'wetravel-cyan-dark': '#00B8A9',
        'wetravel-green': '#4ADE80',
      }
    }
  }
}
```

## 布局规范

### 页面结构

```html
<!-- 后台页面基础结构 -->
<div class="min-h-screen bg-gray-50">
  <!-- 侧边栏 -->
  <aside class="fixed w-64 h-screen bg-white border-r">
    <!-- 导航菜单 -->
  </aside>
  
  <!-- 主内容区 -->
  <main class="ml-64 p-6">
    <!-- 页面标题 -->
    <div class="mb-6">
      <h1 class="text-2xl font-bold text-gray-900">页面标题</h1>
    </div>
    
    <!-- 内容卡片 -->
    <div class="bg-white rounded-lg shadow p-6">
      <!-- 卡片内容 -->
    </div>
  </main>
</div>
```

### 间距规范

| 用途 | 类名 | 像素 |
|------|------|------|
| 页面内边距 | `p-6` | 24px |
| 卡片内边距 | `p-4` / `p-6` | 16px / 24px |
| 元素间距 | `space-y-4` | 16px |
| 紧凑间距 | `space-y-2` | 8px |

### 卡片样式

```html
<!-- 标准卡片 -->
<div class="bg-white rounded-lg shadow p-6">
  <h2 class="text-lg font-semibold text-gray-900 mb-4">卡片标题</h2>
  <p class="text-gray-600">卡片内容</p>
</div>

<!-- 带边框卡片 -->
<div class="bg-white rounded-lg border border-gray-200 p-4">
  <!-- 内容 -->
</div>
```

## 组件样式

### 按钮

#### 主要按钮

```html
<button class="bg-wetravel-cyan hover:bg-wetravel-cyan-dark text-white 
               px-4 py-2 rounded-lg font-medium transition-colors">
  保存
</button>
```

#### 次要按钮

```html
<button class="bg-white border border-gray-300 text-gray-700 
               hover:bg-gray-50 px-4 py-2 rounded-lg font-medium">
  取消
</button>
```

#### 危险按钮

```html
<button class="bg-red-600 hover:bg-red-700 text-white 
               px-4 py-2 rounded-lg font-medium">
  删除
</button>
```

#### 按钮尺寸

| 尺寸 | 类名 |
|------|------|
| 小 | `px-3 py-1.5 text-sm` |
| 中（默认） | `px-4 py-2` |
| 大 | `px-6 py-3 text-lg` |

### 表格

```html
<table class="min-w-full divide-y divide-gray-200">
  <thead class="bg-gray-50">
    <tr>
      <th class="px-6 py-3 text-left text-xs font-medium text-gray-500 uppercase">
        列标题
      </th>
    </tr>
  </thead>
  <tbody class="bg-white divide-y divide-gray-200">
    <tr class="hover:bg-gray-50">
      <td class="px-6 py-4 text-sm text-gray-900">
        单元格内容
      </td>
    </tr>
  </tbody>
</table>
```

### 状态标签

```html
<!-- 成功 -->
<span class="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium bg-green-100 text-green-800">
  已付款
</span>

<!-- 待处理 -->
<span class="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium bg-yellow-100 text-yellow-800">
  待支付
</span>

<!-- 失败 -->
<span class="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium bg-red-100 text-red-800">
  已取消
</span>
```

### 导航菜单

```html
<!-- 激活状态 -->
<a class="flex items-center px-4 py-2 text-wetravel-cyan bg-cyan-50 
          rounded-lg font-medium">
  <svg class="w-5 h-5 mr-3">...</svg>
  菜单项
</a>

<!-- 默认状态 -->
<a class="flex items-center px-4 py-2 text-gray-600 hover:text-gray-900 
          hover:bg-gray-50 rounded-lg">
  <svg class="w-5 h-5 mr-3">...</svg>
  菜单项
</a>
```

## 表单控件

### 文本输入

```html
<div>
  <label class="block text-sm font-medium text-gray-700 mb-1">
    标签名称
  </label>
  <input type="text" 
         class="w-full px-3 py-2 border border-gray-300 rounded-lg 
                focus:ring-2 focus:ring-wetravel-cyan focus:border-wetravel-cyan
                placeholder-gray-400">
</div>
```

### 下拉选择

```html
<select class="w-full px-3 py-2 border border-gray-300 rounded-lg 
               focus:ring-2 focus:ring-wetravel-cyan focus:border-wetravel-cyan">
  <option>选项 1</option>
  <option>选项 2</option>
</select>
```

### 复选框

```html
<label class="flex items-center">
  <input type="checkbox" 
         class="w-4 h-4 text-wetravel-cyan border-gray-300 rounded 
                focus:ring-wetravel-cyan">
  <span class="ml-2 text-sm text-gray-700">选项文字</span>
</label>
```

### 单选按钮

```html
<label class="flex items-center">
  <input type="radio" 
         class="w-4 h-4 text-wetravel-cyan border-gray-300 
                focus:ring-wetravel-cyan">
  <span class="ml-2 text-sm text-gray-700">选项文字</span>
</label>
```

### 表单验证样式

```html
<!-- 错误状态 -->
<input class="border-red-500 focus:ring-red-500 focus:border-red-500">
<p class="mt-1 text-sm text-red-600">错误提示信息</p>

<!-- 成功状态 -->
<input class="border-green-500 focus:ring-green-500 focus:border-green-500">
```

### 必填标记

```html
<label class="block text-sm font-medium text-gray-700 mb-1">
  字段名称 <span class="text-red-500">*</span>
</label>
```

## 图标系统

### Heroicons 使用

项目使用 Heroicons (https://heroicons.com/)。

```html
<!-- Outline 图标 (推荐用于导航) -->
<svg class="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
  <!-- path data -->
</svg>

<!-- Solid 图标 (用于强调) -->
<svg class="w-5 h-5" fill="currentColor" viewBox="0 0 24 24">
  <!-- path data -->
</svg>
```

### 常用图标尺寸

| 用途 | 类名 |
|------|------|
| 导航菜单 | `w-5 h-5` |
| 按钮内图标 | `w-4 h-4` |
| 大图标 | `w-6 h-6` |
| 状态图标 | `w-5 h-5` |

### 图标颜色

图标颜色跟随文字颜色，使用 `currentColor`：

```html
<a class="text-gray-600 hover:text-wetravel-cyan">
  <svg class="w-5 h-5" fill="none" stroke="currentColor">...</svg>
</a>
```

## 响应式设计

### 断点

| 断点 | 宽度 | 用途 |
|------|------|------|
| `sm` | 640px | 手机横屏 |
| `md` | 768px | 平板 |
| `lg` | 1024px | 笔记本 |
| `xl` | 1280px | 桌面 |

### 常用响应式模式

```html
<!-- 网格布局 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <!-- 卡片 -->
</div>

<!-- 隐藏/显示 -->
<div class="hidden md:block">桌面可见</div>
<div class="md:hidden">移动端可见</div>
```
